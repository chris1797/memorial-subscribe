# 온라인 추모관 구독 서비스

기존 진행했던 프로젝트를 JOOQ → JPA, maria → postgre로 마이그레이션 하며 비즈니스 로직들을 리팩토링하는 사이드 프로젝트

<br/>

Stack
---
- SpringBoot 2. + jdk 17
- jooq 3.14.
- mariaDB
- Vue3 + Vuex

<br/>

사용된 기능들
---
- SpringSecurity-JWT
- IOS/AOS 인앱결제 모듈 구현
- 스케줄링 FCM 푸시
- 서버사이드 동영상 썸네일 추출
- 달력 구현
